Python Installation: Windows

  1. Download the learner files (zip)
  2. Extract the zip
  3. Install the latest version of Python 3.x: https://www.python.org/downloads
  4. Install Pycharm Educational Edition: https://www.jetbrains.com/pycharm-educational
  5. Run Pycharm and open the project folder you unzipped
  6. At the bottom-left click on "Python Console". After a minute the version number should appear.

  7. You're good to go!
